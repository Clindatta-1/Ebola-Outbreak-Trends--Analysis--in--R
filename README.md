## Overview

Ebola virus disease (EVD) remains one of the most lethal infectious diseases known to public health. This project provides a structured, reproducible R analysis of two major modern outbreaks:

| Outbreak | Countries Affected | Duration | Total Cases | Total Deaths |
|---|---|---|---|---|
| West Africa | Guinea, Liberia, Sierra Leone | Mar 2014 – Feb 2016 | ~28,600 | ~11,300 |
| DRC (North Kivu) | Democratic Republic of Congo | Aug 2018 – Jun 2020 | ~3,470 | ~2,280 |

The analysis tracks cumulative case counts, monthly incidence, and case fatality rates (CFR) across countries and outbreak periods.

## Dataset

The dataset is located at [`data/ebola_trends.csv`](data/ebola_trends.csv) and contains **monthly aggregated surveillance data** compiled from WHO situation reports.

### Schema

| Column | Type | Description |
|---|---|---|
| `date` | `Date` (YYYY-MM-DD) | First day of the reporting month |
| `country` | `character` | Affected country or region |
| `confirmed_cases` | `integer` | Laboratory-confirmed cumulative cases |
| `probable_cases` | `integer` | Probable cumulative cases (clinical diagnosis, no lab) |
| `suspected_cases` | `integer` | Suspected cumulative cases under investigation |
| `total_cases` | `integer` | Sum of confirmed + probable + suspected |
| `deaths` | `integer` | Cumulative deaths (all classifications) |
| `case_fatality_rate` | `numeric` | CFR (%) = deaths / total_cases × 100 |
| `outbreak` | `character` | Outbreak label for faceting and filtering |

### Sample Records

```
date,country,confirmed_cases,probable_cases,suspected_cases,total_cases,deaths,case_fatality_rate,outbreak
2014-03-01,Guinea,49,3,4,56,29,51.79,West Africa 2014-2016
2014-09-01,Liberia,2046,214,68,2328,1224,52.58,West Africa 2014-2016
2019-05-01,DRC (North Kivu),1974,20,185,2179,1333,61.18,DRC 2018-2020
```

### Notes on Data Quality

- Cumulative figures are monotonically increasing by design; use `lag()` or `diff()` to derive period-level incidence.
- CFR figures should be interpreted cautiously during active outbreaks due to incomplete outcome ascertainment.
- Suspected cases that were later ruled out are not retroactively removed in this dataset, consistent with original WHO reporting conventions.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/ebola-r-project.git
cd ebola-r-project
```

### 2. Create the figures directory

```bash
mkdir figures
```

### 3. Run the analysis

Open `analysis.R` in RStudio or run it from the terminal:

```bash
Rscript analysis.R
```

Output figures are saved to the `figures/` directory at 300 DPI.

## Analysis Highlights

The script (`analysis.R`) produces three publication-ready visualisations:

### 1. Cumulative Cases Over Time
Faceted line chart comparing total cumulative case counts by country across both outbreak periods, highlighting the exponential growth phase and subsequent plateau.

### 2. Case Fatality Rate (CFR) Over Time
Trend lines with LOESS smoothing illustrating how the CFR evolved throughout each outbreak — reflecting changes in clinical capacity, treatment protocols, and surveillance completeness.

### 3. Monthly New Cases — West Africa (2014–2016)
Grouped bar chart showing monthly incidence by country, revealing the peak transmission window (September–December 2014) and the asymmetric decline across Guinea, Liberia, and Sierra Leone.

## Key Findings

- **Sierra Leone** reported the highest cumulative case count during the West Africa outbreak (~14,100 total cases), while **Liberia** experienced the steepest single-month surge (September 2014).
- **Guinea**, where the outbreak originated, maintained a persistently high CFR (≥ 60%) throughout, contrasting with Sierra Leone's lower CFR (~28–30%) — likely reflecting differences in healthcare capacity and case ascertainment.
- The **DRC 2018–2020** outbreak had a significantly higher average CFR (~61%) than the West Africa epidemic (~44%), consistent with the security-compromised response environment in North Kivu.
- Both outbreaks exhibit a characteristic **exponential growth → plateau** curve, consistent with the effect of ring vaccination and contact tracing programmes.
